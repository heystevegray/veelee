## Dropbox

Convert Dropbox shared links to serve user content directly.

### Bad

https://`www.dropbox.com`/scl/fi/vbzkm68k0td7wk3u5vsbl/hello.mp4

### Good

https://`dl.dropboxusercontent.com`/scl/fi/vbzkm68k0td7wk3u5vsbl/hello.mp4
